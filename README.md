# ae-night-sky-generator
 Generate a night sky in AE

To generate, call: generateNightSky(numStars, sizeVariation, glowRGB);

// recommended ranges
// numStars [50-1000]
// sizeVariation [0-50]
// glowRGB [anyColour]

Example (generateNightSky(5000, 10, [.3176, .8475, 1]);)
